      Sorközi elemek

      Billentyűkombinációk esetén a <kbd> html elem használandó: Mentéshez használjuk a CTRL + S billentyűkombinációt.

      Egy program minta kimenetének jelölésére a <samp> html elem használandó: Ez egy program által jelzett eredmény: 42.

      Amennyiben valamilyen változókat akarunk megjelölni, a <var> html elemre lesz szükségünk: y = mx + b.

      Amennyiben egy bekezdésen belül szeretnénk egy mondatot megjelölni, kiemelni, úgy használjuk <mark> html elemet. Ez a mondat kiemelésre került. Ez pedig nem került kiemelésre.
