Sorközi elemek (Alcím 2)

A kód típusú elemek <code> html elemként viendőek fel. Ennek megfeleőn akár utasítások is megjelölhetőek: SELECT * FROM tabla;

Billentyűkombinációk esetén a <kbd> html elem használandó: Mentéshez használjuk a CTRL + S billentyűkombinációt.

Egy program minta kimenetének jelölésére a <samp> html elem használandó: Ez egy program által jelzett eredmény: 42.

Amennyiben valamilyen változókat akarunk megjelölni, a <var> html elemre lesz szükségünk: y = mx + b.

Amennyiben egy bekezdésen belül szeretnénk egy mondatot megjelölni, kiemelni, úgy használjuk <mark> html elemet. Ez a mondat például most kielemésre került. Ez pedig nem került kiemelésre.
